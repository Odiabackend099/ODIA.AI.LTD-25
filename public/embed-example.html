How to Embed on Your Website

Add this code to your website's HTML:

<!-- Configure API keys (optional, can use environment variables) -->
<script>
window.ODIA_WIDGET = {
  groqApiKey: 'YOUR_GROQ_API_KEY',      // Optional if set in environment
  minimaxApiKey: 'YOUR_MINIMAX_API_KEY', // Optional if set in environment
  minimaxGroupId: 'YOUR_MINIMAX_GROUP_ID', // Optional if set in environment
  theme: 'light' // or 'dark'
};
</script>

<!-- Load the widget -->
<iframe 
  src="https://odiadev-web-2025-lunc-3l36edbao-odia-backends-projects.vercel.app/widget"
  style="position: fixed; bottom: 24px; right: 24px; width: 96px; height: 96px; border: none; z-index: 1000;"
  title="ODIADEV Voice AI Chat">
</iframe>
Note: The widget automatically loads configuration from environment variables if not provided via window.ODIA_WIDGET.
